# Clean Context Menu package

Cleans up the context menus when you find they are getting too large.

## Features
Allows you to move similar items into sub-menus or hide them completely.

### Supported Groups
 - Tabs: Split Panes
    - Split Left
    - Split Right
    - Split Up
    - Split Down
 - Editor: Split Panes
    - Split Left
    - Split Right
    - Split Up
    - Split Down
 - Tree: Split Panes
    - Split Left
    - Split Right
    - Split Up
    - Split Down
 - Tabs: Close...
    - Close Other Tabs
    - Close Tabs To The Right
    - Close Saved Tabs
    - Close All Tabs
 - Close Pane (single item)
